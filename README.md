
# Backend Framwork Adonis Js With Docker

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# production
$ npm run start

# build
$ npm run build
```

## Test

```bash
# unit tests
$ npm run test

# eslint
$ npm run lint

# typecheck
$ npm run typecheck
```

## Docker
```bash
# build docker mode log
$ docker-compose up

# build docker not mode log
$ docker-compose up -d

$ ndocker-compose down

```
